# 🎸 archie's songwriting suite 

Welcome to **archie's songwriting suite**—a hyper-focused, dopamine-infused, all-in-one web app designed to help you construct brutal breakdowns, map out chaotic poly-meters, and architect massive metalcore riffs. 

Built entirely with pure, vanilla web technologies and zero external dependencies, this suite combines slick, deep-space aesthetics with advanced music theory utilities. 🧠✨

---

## 🎨 Inside the Suite

The application features four interconnected tools engineered to destroy writer's block and streamline your creative hyperfixations:

### 1. ⏱️ Breakdown Generator
* **What it does:** Generates mathematically precise rhythm patterns across 2 bars of 4/4 time.
* **The Magic:** Uses a zero-lag scheduler via the **Web Audio API** to snap note divisions (quarter notes, eighths, sixteenths, and rests) into a crystal-clear digital metronome. 
* **Visuals:** Features a smooth, glowing playhead engine tracking your position across the staff in real-time.

### 2. 🌀 Poly-Meter Engine
* **What it does:** Visualizes and plays complex, overlapping time signatures (e.g., 7 against 4) simultaneously.
* **The Magic:** Tracks an Outer Ring (metronome click) against an Inner Ring (heavy kick drum pulse).
* **Visuals:** Uses HTML5 Canvas to map dynamic orbital nodes that rotate and collide flawlessly according to the math of your subdivision.

### 3. 🎼 Theory Fretboard
* **What it does:** Displays an interactive guitar fretboard mapped out across 15 frets.
* **The Magic:** Dynamically updates every note based on your chosen tuning (**Drop C**, **Drop B**, or **Drop A**) and scale (**Phrygian Dominant**, **Harmonic Minor**, or **Natural Minor**). 
* **Visuals:** Includes custom string thicknesses, fretboard inlay dots (including the double 12th fret marker), and distinctly highlights root notes in aggressive dark red.

### 4. 🎹 Chord Architect
* **What it does:** Instantly cooks up massive, atmospheric chord progressions rooted in Roman Numeral music theory.
* **The Magic:** Automatically transposes progressions based on the lowest note of your active guitar tuning so your chords are always perfectly playable.

## ⚡ Quick Launch (Get Started in 3 Seconds!)

Because this app is completely self-contained in a single file, getting it up and running is stupidly simple:

1. **Clone or Download** this repository.
2. Double-click **`index.html`** to open it instantly in any modern web browser.
3. Turn your volume up, lock into a tempo, and start writing! 🚀

## 🛠️ Built With

No bloated frameworks or heavy libraries here. This suite is lean, mean, and lightning-fast:

* **HTML5:** semantic layout structure.
* **CSS3:** Designed with a custom *Google Gemini Space* theme (deep pitch blacks, clean elevated cards, and glowing blue/red neon accent colors).
* **Vanilla JavaScript:** Powers the state sync and modal management.
* **Web Audio API:** Drives the synthesized metronome engine for zero lag or timing drift.
* **HTML5 Canvas API:** Handles smooth, high-DPI rendering for notation and orbital physics animations.
